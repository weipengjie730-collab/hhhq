# 事实核查参考

当建议涉及当前事实、政策、考试、就业市场、平台规则、薪资、签证、学校申请要求时，必须先核查再输出。

## 优先来源

- 官方部门：教育部、人社部、国家公务员局、各省人事考试网、各高校招生网
- 官方报名/招录系统：研招网、国家公务员考试录用系统、各省公务员/事业单位系统
- 平台规则：小红书、抖音、B 站、微信公众平台等官方创作者规则
- 企业招聘：公司官网、校招官网、可信招聘平台的具体岗位描述
- 留学申请：目标学校官网、项目页面、签证/移民官方页面

## 输出要求

- 区分“事实”“推断”“建议”。
- 不使用单一网文概括整个行业。
- 不把全国平均竞争比直接套到具体学生身上。
- 不给出未经核查的薪资承诺、上岸率承诺或平台变现承诺。
- 若没有可靠来源，要明确说明“不确定，需要进一步核查”。

## 常见需要核查的问题

- 某专业能否报考某公务员/事业编岗位
- 某考试的报名时间、资格限制、录取/招录人数
- 某学校项目的申请要求、学费、截止日期
- 某平台创作者规则、分成规则、商业化门槛
- 某岗位是否真实要求研究生学历、证书或实习经验
