# 大学生奥德赛 Skill

一个面向中国大学生的 Codex skill，用来帮助学生在考研、考公之外，系统比较就业、自媒体、创业、出国、技能转型等路径，并输出可执行的 7/30/90 天行动计划。

它不是“劝退考研考公”的工具，而是帮助学生避免把默认选项误当成唯一选项。

## 适合场景

- 大学生不知道毕业后该考研、考公还是就业
- 想做自媒体、创业或转行，但不知道怎样验证
- 已经决定考研/考公，但缺少失败后的备选方案
- 想从目标职业反推大学期间该学什么、做什么项目、投什么实习
- 家庭期待、同辈压力和个人兴趣发生冲突，需要更清醒地比较路径

## 使用方式

把 `大学生奥德赛.skill` 文件夹复制到你的 Codex skills 目录中，例如：

```bash
~/.codex/skills/大学生奥德赛.skill
```

然后新开一个 Codex 会话。可以这样调用：

```text
使用 $college-odyssey-guide 帮我比较大学毕业后的路径，并制定 7/30/90 天行动计划。
```

> 说明：`college-odyssey-guide` 是内部调用名，界面显示名是“大学生奥德赛”。

## 设计原则

- 先核查前提，再给建议
- 不羞辱考研、考公、就业、自媒体、创业、出国等任何路径
- 把“真正想要的结果”和“目前只知道的路径”拆开
- 用小规模验证任务代替空泛规划
- 输出必须包含路径比较、风险、备选方案和行动计划
