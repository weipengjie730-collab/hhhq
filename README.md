# Skills 合集

我的 Claude / Cowork skill 开源合集。每个子文件夹是一个独立 skill，可单独取用。

## 已有 skill

| Skill | 说明 |
| --- | --- |
| [`xhs-hotspot-note`](./xhs-hotspot-note) | 把当下热点从"AI 实用向"角度加工成可直接发布的小红书图文笔记（标题/正文/标签/配图建议），可选浏览器半自动填入创作者后台。 |

## 什么是 skill

skill 是一份带 `SKILL.md` 的指令文件夹，告诉 Claude 在特定场景下该怎么做。把文件夹放进 Claude/Cowork 的 skills 目录即可使用。

## 使用说明

每个 skill 的用法见各自文件夹内的 `SKILL.md`。

## 合规与风险提示

`xhs-hotspot-note` 不调用任何第三方/逆向接口。小红书没有面向个人的官方发布 API，发布环节为"内容准备 + 浏览器半自动（最后一步人工点发布）"，请遵守平台规则，自负账号风险。
